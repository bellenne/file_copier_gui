# Копирование файлов с windows сервера

1. Укажите файл со списком имён скаченного из задания в MES
2. Укажите папку на сервере, где хранятся файлы для печат (1 раз, при первом запуске)
3. Укажите папку сохранения файлов
4. Укажите, нужно ли удалить файлы с сервера после удаления
5. Нажмите кнопку "СТАРТ" для того чтобы начать копирование файлов

1. Файлы копируются с тем же именем что и на сервере.
2. Структура подпапок не сохраняется
3. Если в папке назначения есть файлы с тем же именем что и в списке - они перезапишутся
